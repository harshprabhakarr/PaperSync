# 📄 PaperSync

🚀 Built using modern full-stack technologies (Next.js + Supabase)

> A collaborative research and document management platform designed to simplify how users explore, organize, and interact with academic content.

---

## 🚀 Overview

PaperSync is a full-stack web application developed as a team project that helps users discover, manage, and engage with research papers efficiently.
It focuses on improving accessibility, collaboration, and structured workflows in academic environments.

---

## ✨ Key Features

* 📂 Organized document & research management
* 🔍 Easy navigation of academic papers
* 🤝 Collaborative workflow support
* 🧠 Smart recommendation system
* 🔐 Secure backend using Supabase

---

## 🛠️ Tech Stack

### 💻 Frontend

* Next.js
* React.js
* HTML5
* CSS3

### ⚙️ Backend & Database

* Supabase

### 🔧 Languages

* JavaScript
* TypeScript

---

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/harshprabhakarr/PaperSync.git

# Navigate to project folder
cd PaperSync

# Install dependencies
npm install

# Run the project
npm run dev
```
